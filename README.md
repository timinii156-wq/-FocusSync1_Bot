# FocusSyncBot

An accountability-focused Telegram bot designed for collaborative focus sessions.

## Features
- Starts focus sessions via the `/focus` command.
- Tracks user activity during sessions.
- Sends alerts when focus is broken.

## Getting Started
1. Clone the repository.
2. Create a `.env` file with `TELEGRAM_BOT_TOKEN=your_token_here`.
3. Install requirements: `pip install -r requirements.txt`.
4. Run the bot: `python bot.py`.

## Deployment
This bot is designed to be deployed on platforms like Railway using the GitHub repository integration.
